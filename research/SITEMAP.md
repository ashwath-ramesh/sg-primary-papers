# Proposed sitemap (new polished site)

## Top-level
- `/` Home (search-first)
- `/papers` Browse/search
- `/levels/p1` … `/levels/p6`
- `/subjects/english` `/subjects/maths` `/subjects/science` `/subjects/chinese` `/subjects/higher-chinese`
- `/years/2025` etc.
- `/assessments/prelim` `/assessments/wa1` `/assessments/wa2` `/assessments/wa3` `/assessments/eoy`
- `/schools/<slug>`
- `/paper/<id-or-slug>` Paper detail page
- `/collections/<slug>` Curated sets
- `/about` / legal / copyright

## UX flows

### Flow A: Parent wants “P6 Maths 2025 Prelim”
Home → choose Level=P6 → Subject=Maths → Year=2025 → Assessment=Prelim → results list → paper detail → download/link.

### Flow B: Student wants "practice now"
Home → search "fractions" → show *original* practice sets + linked papers.

### Flow C: Teacher wants "Rosyth P6 English 2024"
Home → School=Rosyth → Level=P6 → Subject=English → Year=2024.
