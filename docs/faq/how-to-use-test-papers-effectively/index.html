How to use test papers effectively (without wasting them)

More papers ≠ more improvement. The win is a tight loop: attempt → review → targeted practice → re-test.

The 20–20–20 loop

  1. 20 mins attempt: short section, timed.
  2. 20 mins review: label each mistake (careless / concept / method / wording).
  3. 20 mins fix: do 3–5 similar questions or redo the same section.

What to track (minimal)

  • Topic of mistake
  • Type of mistake
  • One sentence: what to do differently next time
